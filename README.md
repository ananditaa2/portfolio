# Anandita Indurkhya — AI & ML Engineer Portfolio ✦

> **Live Preview:** 🚀 [https://ananditaa2.github.io/portfolio/](https://ananditaa2.github.io/portfolio/)

Welcome to the source code of my personal portfolio. This space is designed to showcase my journey as a developer, designer, and AI/ML enthusiast. The UI strictly adheres to a "Premium Tech" design language, balancing complex interactive animations with robust functionality to reflect a strong intersection of engineering and visual aesthetics.

## ✨ Features
* **Premium Tech Interface:** Deep charcoal & high-contrast neon styling suited for the modern Web3/AI ecosystem. Integrated Dark/Light Mode toggle.
* **Interactive Neural Cursor:** Built-in generative canvas (`Three.js` / HTML5 Canvas) replacing the standard mouse—featuring a dynamic, trailing neural network node connection and ambient light spotlight mapping.
* **Deck-of-Cards Scroll Dynamics:** State-of-the-art parallax and intersection layering utilizing `GSAP ScrollTrigger` allows individual project sections to slide and stack fluidly as the user navigates.
* **Laser-Beam Timeline:** Animated scroll-driven progression lines dynamically rendered matching vertical layout paths on the document.
* **Serverless Email API:** Zero-configuration instant email bridging via `FormSubmit.co` embedded straight into the native `fetch` API.

## 🛠 Tech Stack
* **Core:** HTML5, Vanilla JavaScript, CSS3
* **Libraries:** Three.js, GSAP (ScrollTrigger)
* **Design Pattern:** Functional Glassmorphism, Micro-Animations, Sticky Scroll-Mapping.
